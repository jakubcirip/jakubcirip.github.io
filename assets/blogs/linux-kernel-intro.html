Linux Kernel Intro

The Linux kernel

The Linux kernel is the core component of the Linux operating system. It acts as a bridge between hardware and software, managing system resources, memory, processes, and hardware communication. That is, it is a special program that is responsible for managing low-level functions of the computer.


Contents

In this introductory article, we'll look at what the kernel does, the difference between user space and kernel space, and how developers can begin exploring kernel development.

What is the kernel for?

The Linux kernel consists of 6 different subsystems that enable core functions, which are:
- System call implementation (SCI)
- Process management
- Memory management
- Virtual memory
- Network stack
- Device drivers
